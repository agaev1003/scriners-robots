### Портфельная логика

Стратегия разделена на **два независимых портфеля** по ликвидности:

| Параметр | S1 (ликвидные) | S2 (неликвидные) |
|---|---|---|
| Порог ликвидности | >= 100 млн ₽/день | < 100 млн ₽/день |
| Макс. позиций | 2 | 2 |

**Общий портфель (Combined)** = S1 + S2 (equity суммируется).

Каждый портфель управляется отдельно:

1. **Вход** — по цене закрытия T+1 (после подтверждения VR и отсутствия гэпа). Капитал распределяется пропорционально весу `min(VR/3, 3)`.

2. **Выходы** (проверяются каждый день, приоритет сверху вниз):
   - **Катастрофический стоп** — если low <= EP × (1 − 28%) → выход по стоп-цене
   - **Трейлинг-стоп** — активируется после `tsAfter` дней, стоп = peak × (1 − tsDist%)
   - **Max Hold** — принудительный выход по close

3. **ATR-ротация** — если портфель полон и есть кандидат с ATR >= weakATR × 1.5 и разница >= 0.5 п.п., слабая позиция заменяется.

4. **Дивидендные гэпы** — в дни ex-date стопы не проверяются (используются реальные даты MOEX ISS, при отсутствии — эвристика).
